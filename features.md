# Welcome to the features page
Here is the list of feature in this library
 - ## think <br> 
   this is console.log and is used like <br>
    ```javascript
    think("hello world")'
    ```
 - ## message <br> 
   this is alert and is used like <br>
    ```javascript
    message("hello world")
    ```
 - ## Math <br> 
   these are the math functions and are used like <br>
    ```javascript
    add(x, y)
    // x + y
    sub(x, y)
    // x - y
    mul(x, y)
    // x * y
    div(x, y)
    // x / y
    mod(x, y);
    // x % y
    round(x);
    // math.floor(x)
    power(x, y);
    // Math.pow(y , x);
    randomfrom(x, y);
    // x = min, y = max
    equal(x, y);
    // x == y
    ```
 - ## Question <br>
   this is prompt and is used like
   ```javascript
   question("");
   ```
